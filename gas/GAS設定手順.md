# お問い合わせフォーム（GAS）設定手順

HPのフォーム送信を Google Apps Script（GAS）で受け取り、
mari@takagi-gyosei.jp への通知＋送信者への自動返信を行うための設定手順です。
**一度設定すれば、あとは無料で動き続けます。**

## 1. GASプロジェクトを作る

1. **mari@takagi-gyosei.jp を送信元にしたい場合は、そのGoogleアカウントでログイン**した状態で
   https://script.google.com を開く（メールはスクリプトを作ったアカウントから送信されます）
2. 「新しいプロジェクト」をクリック
3. エディタに最初から入っているコードを全て消し、`contact-form.gs` の中身を貼り付ける
4. 上部のプロジェクト名を「HPお問い合わせフォーム」などに変更して保存（Cmd+S）

## 2. ウェブアプリとして公開する

1. 右上の「デプロイ」→「新しいデプロイ」
2. 歯車アイコン →「ウェブアプリ」を選択
3. 次のように設定する
   - 説明：任意（例：お問い合わせフォーム）
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**（←重要。これ以外だとHPから送信できません）
4. 「デプロイ」→ 初回はアクセス承認を求められるので、
   「アクセスを承認」→ アカウント選択 →「詳細」→「（プロジェクト名）に移動」→「許可」
5. 表示された **ウェブアプリのURL（https://script.google.com/macros/s/～/exec）をコピー**

## 3. HPにURLを設定する

1. `contact.html` をエディタで開く
2. `<form class="form-card reveal" id="contactForm" data-gas="">` の
   `data-gas=""` の中に、コピーしたURLを貼り付ける

   例：
   `<form class="form-card reveal" id="contactForm" data-gas="https://script.google.com/macros/s/AKfyc.../exec">`

3. 保存してブラウザで開き、テスト送信する
   - mari@takagi-gyosei.jp に通知メールが届く
   - フォームに入力したメールアドレスに自動返信が届く
   - 画面に「送信が完了しました」のモーダルが出る

## 補足

- **コードを修正したら**「デプロイ」→「デプロイを管理」→ 鉛筆アイコン →
  バージョン「新バージョン」→「デプロイ」で反映されます（URLは変わりません）
- 送信件数の目安：無料のGoogleアカウントは1日100通、Google Workspaceは1日1500通まで
- 迷惑ボット対策として、画面に見えない入力欄（ハニーポット）を仕込んであります
